# tobybessant.co.uk

Personal portfolio site built with [Hugo](https://gohugo.io/), hosted on GitHub Pages.

## Development

```bash
hugo server        # live-reload at http://localhost:1313
hugo --minify      # production build → public/
```

## Content

Content lives in `data/*.yaml`. Edit a file and push to deploy.

## Deployment

Pushes to `main` build and deploy via GitHub Actions.
